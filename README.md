# IRA Apprenticeship Compliance Dashboard

Interactive Tableau dashboard tracking federal IRA apprenticeship 
ratio requirements for construction and renewable energy workforce 
management. Enables project managers and compliance teams to monitor 
weekly ratios, identify violations, and drill down to craft and 
employee-level detail.

## 🎯 Live Dashboard
[(https://public.tableau.com/views/ApprenticeshipComplianceDashboard/IRADashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)]

## 📊 Key Features
- Federal apprentice/journeyman ratio tracking
- Weekly violation flagging  
- Multi-tab drill-down architecture
- Craft-level and employee-level detail views
- Dynamic filtering (project, week, classification, craft)

## 🛠️ Skills Demonstrated
- **Tableau:** Calculated fields, drill-down navigation, dashboard UX
- **SQL:** CTEs, window functions, multi-table joins (Snowflake)
- **Data Modeling:** Workforce hours aggregation, ratio calculations
- **Compliance Reporting:** Federal IRA apprenticeship requirements
